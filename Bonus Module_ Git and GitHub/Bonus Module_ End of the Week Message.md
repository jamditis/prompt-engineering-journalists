# Bonus Module: Git and GitHub for journalists

## End of the week message

You made it through the bonus module.

This week you learned how Git tracks changes, what commits and branches are, and how GitHub works as a shared workspace. More importantly, you learned when and why to ask Claude Code to use these tools on your behalf.

**What you learned:**
- Git's three areas: working directory (where you edit), staging area (where you prepare), repository (where history lives)
- Commits are snapshots of your project at meaningful checkpoints
- Branches let you experiment without risking working code
- GitHub is where your project lives online — for backup, collaboration, and publishing
- Pull requests are how teams review changes before they're merged
- Claude Code handles the Git commands; your job is knowing what to ask for and when
