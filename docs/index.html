Move from chat windows to the command line

Learn to use AI coding assistants in your terminal. Write prompts that generate working code, automate repetitive tasks, and build tools for your newsroom.

Get started View syllabus

Course modules

Five weeks covering the fundamentals of CLI-based AI tools for journalism.

  1. 1

    Getting started with CLI tools

    Install Node.js, set up your terminal, and run your first AI-assisted coding session. Understand how command-line tools differ from chat interfaces.

  2. 2

    Writing effective prompts

    Structure prompts for code generation. Learn to provide context, specify constraints, and iterate on outputs. Build a simple data-processing script.

  3. 3

    Working with your codebase

    Use AI assistants to navigate and modify existing projects. Edit files, refactor code, and maintain version control with Git.

  4. 4

    Building journalism tools

    Apply your skills to journalism workflows. Scrape public data, clean datasets, generate visualizations, and automate routine tasks.

  5. 5

    Final project

    Build a complete tool for your newsroom. Document your process, share your work, and get feedback from peers.

Quick start

Get up and running in three steps.

1

Install Node.js

Download from nodejs.org and run the installer. This gives you npm, the package manager you'll use to install AI tools.

node --version
2

Install an AI tool

Pick one to start. Gemini CLI is free and works well for beginners.

npm install -g @anthropic-ai/claude-code
# or
npm install -g @google/gemini-cli
3

Run your first prompt

Open your terminal, navigate to a project folder, and start the tool.

cd ~/my-project
claude
# or
gemini

Tools covered

We focus on four AI coding assistants. You only need one to complete the course.

Claude Code

Anthropic's terminal-based assistant. Strong at reasoning through complex tasks and working with existing codebases.

Requires API key or subscription

Documentation

Gemini CLI

Google's command-line tool. Free tier available. Good for quick scripts and data processing tasks.

Free tier with Google account

Documentation

Aider

Open-source pair programming tool. Works with multiple AI providers. Excellent Git integration.

Open source, bring your own API key

Documentation

Codex CLI

OpenAI's terminal assistant. Integrates well with existing OpenAI workflows and API keys.

Requires OpenAI API key

Documentation
