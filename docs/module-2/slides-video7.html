Apps vs harnesses
Try that in a chat window
Browser chat

ChatGPT, Claude.ai, Gemini

  • No filesystem access
  • Can't install tools or packages
  • Can't run scripts or shell commands
  • One conversation = one task
  • Copy-paste everything manually
CLI harness

Claude Code, Gemini CLI

  • Full filesystem read/write
  • Installs packages as needed
  • Runs scripts, coordinates tools
  • Multi-step pipelines in one session
  • Reads and writes thousands of files

Skills vs commands
Two flavors, one format
Skill

Domain knowledge applied with judgment

Like a style guide — reference material, contextual interpretation. The AI decides how to apply the instructions based on the situation.

Command

Deterministic trigger, same steps every time

Like a pre-publication checklist — fixed order, no skips. Same input, same sequence of actions, same output format.

Both are .md files in .claude/commands/ — filename = slash command

Two types of hooks
Notify vs. stop
Notify hook

The automated second reader

  • Flag issues without stopping your work
  • Unattributed quotes, AI filler language, style violations
  • Work continues — you review flags later
  • Like a copy editor marking up your draft while you keep writing
Stop hook

The one-way door guardian

  • Pause before irreversible actions
  • Deleting files, pushing code, publishing content
  • "This is about to do X — proceed?"
  • Editorial rule: does a human need to see this before it goes out?

Pain points → solutions
What the skill encodes
Rate limits

Platforms throttle downloads

Skill adds delays between requests, retries on 429 errors, rotates user agents. You learned this the hard way once — the skill remembers forever.

Audio quality

Some transcripts come back garbage

Skill falls back to a local Whisper model when API results have low confidence. Flags files that need manual review.

Platform blocks

TikTok fights scraping

Skill uses yt-dlp with cookie support and platform-specific extractors. Encodes the workaround so you don't rediscover it.

Inconsistent output

Every run produced a different folder layout

Skill enforces a standard directory structure, naming conventions, and output manifest. Reproducible every time.
