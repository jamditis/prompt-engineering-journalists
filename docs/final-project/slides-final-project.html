Requirements
Your project must meet four criteria
1

Use a CLI AI tool

Claude Code, Gemini CLI, Codex CLI, or another CLI tool with instructor approval

2

Include automation

Go beyond a single prompt: a reusable script, multi-stage pipeline, scheduled task, or custom skill

3

Solve a real problem

Address an actual challenge: something you do manually, avoid because it's tedious, or that needs better consistency

4

Include documentation

A README explaining the problem, setup, usage, expected output, and limitations

Submit a GitHub repository or zip file with all code, config files, a README, and sample input/output.

Inspiration
Suggested project ideas

Newsletter pipeline

Gather beat content, summarize developments, draft newsletter sections as reusable weekly output

Transcript processor

Extract key quotes, identify themes, generate structured research documents from interview audio

Source verification

Encode your fact-checking process into a skill: check claims, document sources, flag what needs confirmation

Social content generator

Take a published story and produce platform-specific posts, threads, and newsletter blurbs

Meeting notes processor

Extract action items and newsworthy decisions from city council, school board, or public meetings

Public records pipeline

Clean and standardize messy public records (PDFs, spreadsheets, text files) for analysis

Custom beat skill

Build a specialized skill for your coverage area with domain knowledge and recurring task automation
