Why harnesses matter

Same model, different capabilities

Chat window

  • Conversational sandbox
  • No file access
  • Copy-paste workflow
  • No persistence between sessions

CLI harness

  • Real machine access
  • File I/O and editing
  • Automation and scripting
  • Version control built in

The model is the same — the difference is what it can reach.
