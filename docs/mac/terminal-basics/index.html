MAC · FOLLOWING ALONG ON MACOS

Terminal basics for Mac users

Joe's Module 1 demos run in PowerShell on Windows. On a Mac, you do the same work in a program called Terminal. The commands are almost identical. Here's what changes and what doesn't.

SHORT ANSWER

Open Terminal, type claude, press Enter. That's the same first step Joe takes on Windows, just in a different program. The cd, ls, and mkdir commands you'll see him type all work exactly the same way in macOS Terminal.

Where Terminal lives on your Mac

Terminal is an app that ships with every Mac. There are three easy ways to open it:

  • ›Spotlight. Press ⌘ Space, type terminal, press Enter. Fastest way in.
  • ›Finder. Open the Applications folder, then the Utilities subfolder. Terminal is in there.
  • ›Launchpad. Open Launchpad, find the Other folder, open it, and Terminal is inside.

When Terminal opens you'll see a window with a single line of text that looks something like this:

yourname@MacBook-Pro ~ %

That's a prompt. It's waiting for you to type something. The ~ (tilde) is a shortcut for "your home folder." The % at the end is where your cursor will sit. When you see the prompt, you're ready to type a command.

The commands you'll actually need for Module 1

Joe's video walks through a specific sequence: create a folder, move into it, launch Claude Code. Here's the macOS version of each step, with the exact commands you'd type.

Make a folder for your course work

mkdir mooc

mkdir means "make directory." It creates a folder called mooc inside wherever you are right now. If you just opened Terminal, that's your home folder.

Step into that folder

cd mooc

cd means "change directory." Think of it as double-clicking a folder — you're now inside it. Your prompt will update to show the folder name so you always know where you are.

See what's in the current folder

ls

ls means "list." Shows you every file and folder in the folder you're currently in. Use this anytime you lose track of what's around you.

Check where you are

pwd

pwd stands for "print working directory." Shows the full path of the folder you're in right now. Helpful when the prompt is ambiguous or you've been navigating for a while.

Go back up one folder

cd ..

The two dots mean "the folder that contains this one." It's the Terminal version of clicking the back arrow in Finder.

Launch Claude Code

claude

This is the same command Joe types in the video. Claude Code starts an interactive session inside whatever folder you're currently in. From this point forward you're not typing terminal commands anymore — you're talking to Claude, and Claude is running commands for you. To exit a Claude session, type /quit or press Ctrl C twice.

PowerShell to Terminal, command by command

Everything in Joe's Module 1 walkthrough translates directly. Here's the cheat sheet:

WHAT JOE TYPES (POWERSHELL) WHAT YOU TYPE (TERMINAL)
cd ~/Desktop cd ~/Desktop
mkdir MOOC mkdir MOOC
cd MOOC cd MOOC
ls ls
claude claude
Clear screen: cls Clear screen: clear
Show file contents: Get-Content file.md Show file contents: cat file.md
Delete file: Remove-Item file.md Delete file: rm file.md

The top five commands are identical on both platforms. The bottom three are where PowerShell and macOS Terminal differ slightly — you probably won't need any of them for Module 1, but they're worth knowing.

A faster way to get into a folder

Sharon Machlis shared this tip in the Technical Questions forum, and it's a keeper. macOS has a built-in way to open Terminal already inside a specific folder, so you don't have to type cd commands to get there:

  • 1.Open Finder and find the folder you want to work in.
  • 2.Right-click (or two-finger click) the folder.
  • 3.Hover over Services and pick New Terminal at Folder.

If you don't see that option, it's probably turned off in System Settings. Go to System Settings → Keyboard → Keyboard Shortcuts → Services → Files and Folders and check the box next to New Terminal at Folder. Takes ten seconds and saves typing every time after that.

You can also drag a folder from Finder into an open Terminal window and it'll paste the full path — handy when you want to cd into a folder that's nested deep in your Documents.

Pasting, copying, and stopping a command

  • ›Paste: ⌘ V. Same as everywhere else on macOS. Works inside a Claude Code session too.
  • ›Copy from the terminal: select the text with your mouse, then ⌘ C.
  • ›Stop a running command: Control C (that's the literal Control key, not Command). This interrupts whatever the terminal is currently doing.
  • ›Exit Claude Code: type /quit, or press Control C twice in a row.
  • ›Close the Terminal window: ⌘ W or just click the red dot.

IF "CLAUDE" SAYS "COMMAND NOT FOUND"

You've installed Claude Code but the claude command doesn't work. This is a PATH problem — your shell doesn't know where Claude Code was installed. See the Fix PATH after install page for the fix. It's usually a one-line change to your shell config.
