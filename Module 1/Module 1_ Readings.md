# Module 1: Escaping the chat window

## Required readings

### 1. Official documentation (pick one tool to start)

Choose the tool you plan to install and read its getting started guide:

- **Claude Code:** https://docs.anthropic.com/en/docs/claude-code
  - Anthropic's official CLI. Requires an Anthropic API key or Claude subscription.

- **Gemini CLI:** https://github.com/google-gemini/gemini-cli
  - Google's command-line interface. Free tier includes 1,000 requests per day.

- **OpenAI Codex CLI:** https://github.com/openai/codex
  - OpenAI's terminal tool. Requires ChatGPT Plus subscription.

- **Aider:** https://aider.chat/docs/install.html
  - Open source tool that works with multiple LLM providers. Pay-per-use pricing.

### 2. Why the command line matters for journalists

- Simon Willison, "CLI tools are having a renaissance"
  - https://simonwillison.net/2024/Mar/22/cli-tools/
  - Explains why experienced developers are moving back to terminal-based workflows.

- Ethan Marcotte, "AI and the command line"
  - https://ethanmarcotte.com/wrote/ai-and-the-command-line/
  - Practical perspective on when terminal tools beat web interfaces.

### 3. Terminal basics (if needed)

If you are new to the command line:

- Ubuntu's "Using the terminal"
  - https://ubuntu.com/tutorials/command-line-for-beginners
  - Works for Mac and Linux users. Windows users should use Git Bash or WSL.

- The Odin Project, "Command Line Basics"
  - https://www.theodinproject.com/lessons/foundations-command-line-basics
  - Interactive tutorial covering navigation and basic commands.

- Tool documentation: 15-20 minutes

- Terminal basics (if needed): 30-45 minutes
